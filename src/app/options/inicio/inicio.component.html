Ver instrucciones

¿Cómo usar la aplicación?
  • En la pestaña Analizar SQL, puedes escribir código o subir un archivo .sql.
  • Se admiten consultas como INSERT INTO, SELECT INTO. Además soporta que contengan la clausula JOIN (LEFT, RIGHT, INNER).
  • No usar paralabras compuestas, usar ; al final de las consultas y en formato PascalCase
  • Después de procesar, consulta la opción Estructura de Datos. El resultado se vera reflejado en "Dependencias" y "Diagrama".
  • Guarda tu linaje para consultas futuras, esto lo podrás visualizar desde la opción "Historial".
